Etiquetas HTML Comunes y Avanzadas

Esta página muestra diversas etiquetas HTML, explicando su función y cómo se escriben.

Navegación (nav)

La etiqueta nav se usa para definir una sección de navegación.

<nav>
    <!-- Contenido de navegación -->
</nav>

Etiquetas de Texto

Encabezados (h1 a h6)

Los encabezados se usan para definir la estructura y jerarquía del contenido. 

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

Esto es un h1

Esto es un h2

Esto es un h3

Párrafos (p)

La etiqueta p se usa para definir párrafos de texto.

<p>Este es un párrafo de ejemplo.</p>

Este es un párrafo de ejemplo.

Texto enfatizado (em) y fuerte (strong)

<em>Texto enfatizado</em>
<strong>Texto fuerte</strong>

Esto es texto enfatizado y esto es texto fuerte.

Etiquetas Multimedia

Imágenes (img)

La etiqueta img se usa para insertar imágenes.

<img src="ruta/a/la/imagen.jpg" alt="Texto alternativo">

Video (video)

La etiqueta video se usa para insertar videos.

<video controls>
    <source src="ruta/al/video.mp4" type="video/mp4">
    Tu navegador no soporta el elemento de video.
</video>

Audio (audio)

La etiqueta audio se usa para insertar archivos de audio.

<audio controls>
    <source src="ruta/al/audio.mp3" type="audio/mpeg">
    Tu navegador no soporta el elemento de audio.
</audio>

Etiquetas de Formulario

Formulario (form)

La etiqueta form se usa para crear formularios.

<form>
    <!-- Elementos del formulario -->
</form>




Etiquetas Semánticas

Artículo (article)

La etiqueta article se usa para contenido independiente y autónomo. 

<article>
    <h3>Título del artículo</h3>
    <p>Contenido del artículo...</p>
</article>

Título del artículo

Este es un ejemplo de contenido de artículo.

Aside (aside)

La etiqueta aside se usa para contenido relacionado pero separado del contenido principal. 

<aside>
    <h4>Contenido relacionado</h4>
    <p>Información adicional...</p>
</aside>

Contenido relacionado

Esta es información adicional relacionada con el tema principal.

Pie de página (footer)

La etiqueta footer se usa para definir un pie de página para un documento o sección. 

<footer>
    <p>&copy; 2023 Ejemplo de Página HTML</p>
</footer>
